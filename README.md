# Telegram-бот для салона красоты
Готовый Telegram-бот для салонов красоты. Клиент смотрит услуги с фото, выбирает мастера, записывается онлайн.
**Демо:** [@beauty_persona_demo_bot](https://t.me/beauty_persona_demo_bot)
## Возможности
- Каталог услуг с фото и ценами (16 услуг)
- Список мастеров
- Онлайн-запись
- Акции и информация о салоне
- Отзывы с уведомлением администратору
## Стек
Python 3.10+, python-telegram-bot v20+, ConversationHandler
## Быстрый старт
1. `pip install -r requirements.txt`
2. Вставить токен в config.py
3. `python bot.py`
